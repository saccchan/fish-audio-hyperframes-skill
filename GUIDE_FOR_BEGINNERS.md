# Fish Audio × HyperFrames 初心者ガイド

## 🎬 これは何？

**「台本を書くだけで、プロ品質のナレーション付きアニメーション動画が自動で作れるツール」です。**

### できること

✅ 台本を書くだけで、高品質な音声ナレーションを自動生成  
✅ 音声に合わせて文字が表示されるアニメーション動画を作成  
✅ 背景やエフェクトも自動で追加  
✅ YouTube、Instagram、TikTokなどに投稿できる動画ファイルを出力

### こんな時に使えます

- 📱 製品・サービスの紹介動画
- 📚 オンライン講座やチュートリアル動画
- 🎙️ ポッドキャストの視覚化
- 📊 プレゼンテーション動画
- 🌐 多言語対応の説明動画

---

## 🚀 使い方（5ステップで完成）

### ステップ1️⃣ 台本を用意する

動画で話したい内容を、メモ帳などに書きます。

**例: 製品紹介動画の台本**

```
こんにちは、新しい製品をご紹介します。
この製品は、日常生活を便利にする革新的な機能を搭載しています。
使い方はとてもシンプルです。
詳しくは、公式サイトをご覧ください。
今すぐお試しください。
```

**書き方のコツ:**
- 1行 = 1つの場面（シーン）
- 1行は2〜8秒で読める長さに
- 自然な話し言葉で書く

### ステップ2️⃣ Cursorに「動画を作って」と頼む

Cursorを開いて、こんな風に話しかけます:

```
Fish Audioで、製品紹介動画を作ってください。

台本はこちらです:
（台本をコピー&ペースト）

要望:
- 明るくて親しみやすい女性の声
- 背景は青系のグラデーション
- 文字は大きくはっきりと
```

### ステップ3️⃣ AI が自動で音声を生成

AIが台本を読んで、自動的に:
- ✅ プロの声優のような音声を生成
- ✅ 各シーンのタイミングを計算
- ✅ 動画の構成を設計

**あなたがすることは: 何もありません！待つだけ。**

### ステップ4️⃣ プレビューで確認

AIが作った動画をブラウザで確認できます。

「もっと文字を大きく」「色を変えて」など、修正したいことがあれば、Cursorに伝えるだけ。

### ステップ5️⃣ 動画ファイルとして保存

OKなら、Cursorに「動画をレンダリングして」と頼むだけ。

数分後、MP4形式の動画ファイルが完成します！

## 💡 実例で理解する

### 例1: 30秒のプロモーション動画

**台本:**
```
こんにちは、TANRENです。
私たちは、AIでビジネスを加速させます。
生成AIの導入から運用まで、トータルサポート。
まずは無料相談から始めませんか？
今すぐお問い合わせください。
```

**Cursorへの依頼:**
```
Fish Audioで30秒のプロモーション動画を作成してください。

声: 女性、明るく元気な感じ
デザイン: 暗い背景に明るい文字、モダンでプロフェッショナル
エフェクト: 各場面でスムーズに切り替わるように
```

**完成イメージ:**
- 0-6秒: 「こんにちは、TANRENです」（ロゴと共にフェードイン）
- 6-12秒: 「私たちは...」（アイコンが飛び込んでくる）
- 12-20秒: 「生成AIの導入...」（特徴を3つ並べて表示）
- 20-26秒: 「まずは無料相談...」（問い合わせボタンが強調される）
- 26-30秒: 「今すぐお問い合わせください」（ウェブサイトURLを表示）

### 例2: チュートリアル動画（60秒）

**台本:**
```
アプリの使い方を説明します。
ステップ1: まずアプリをダウンロードしてください。
ステップ2: アカウントを作成します。
ステップ3: 好きなトピックを選びます。
これで完了です。さあ、始めましょう！
```

**Cursorへの依頼:**
```
Fish Audioでチュートリアル動画を作ってください。

声: 女性、優しく説明する感じ
デザイン: 明るい背景、ステップごとに番号と絵文字を表示
エフェクト: ステップごとに違う色でハイライト
```

**完成イメージ:**
- 各ステップに大きな番号（1, 2, 3...）
- ステップごとに色が変わる（青→紫→ピンク→黄色）
- アイコンやイラストが動いて説明をサポート

## 🎨 カスタマイズのアイデア

**女性の声:**
- `ja-JP-Female-1`: 明るく元気
- `ja-JP-Female-2`: 落ち着いた大人っぽい

**男性の声:**
- `ja-JP-Male-1`: 力強い

**色の雰囲気:**
```
Cursorに伝える例:
- 「背景をもっと明るくして」

- 「グラデーションを虹色にして」
```

**文字のスタイル:**
```
Cursorに伝える例:
- 「文字を大きくして」
- 「太字にして目立たせて」

```
Cursorに伝える例:

- 「もっとゆっくり切り替わるように」
- 「回転しながら登場するエフェクトを追加」

## 🎯 よくある質問

### Q1: プログラミングの知識は必要？

**A: 全く必要ありません！** Cursorに日本語で話しかけるだけです。

### Q2: 費用はかかる？

**A: Fish Audio APIの利用料が必要です。**
- 料金は使った分だけ（従量課金）
- 30秒の動画なら数十円程度
- 詳しくは[Fish Audio公式サイト](https://fish.audio/)で確認

HyperFrames（動画作成ツール）は無料です。

### Q3: どのくらいの時間で作れる？

**作業時間: 約5〜10分**
- 台本を書く: 3分
- Cursorに依頼: 1分
- AI処理: 2〜5分
- 確認・修正: 2分

**初めてでも30分あれば完成します。**

### Q4: 作った動画はどこで使える？

どこでも自由に使えます！
- YouTube
- Instagram
- TikTok
- 社内プレゼン
- ウェブサイト
- 広告

MP4ファイルなので、どんなプラットフォームでも投稿できます。

### Q5: 音楽（BGM）も追加できる？

**はい！** Cursorに「BGMを追加して」と頼むだけ。

```
例:
「ナレーションの下に、明るいBGMを小さく流して」
```

### Q6: 英語や他の言語でも作れる？

**はい！** 40言語以上に対応しています。

```
Cursorへの依頼例:
「英語版の動画も作って」
「中国語のナレーションで動画を生成」
```

### Q7: 動画の長さに制限はある？

**制限はありません。**
- 短い動画: 15秒（SNS用）
- 標準的: 30秒〜2分（プロモ・紹介）
- 長い動画: 5分以上（チュートリアル・講座）

### Q8: 失敗したらどうする？

**何度でもやり直せます！** 
Cursorに「もう一度最初から作って」と言うだけ。

## 🌟 プロからのアドバイス

### 台本作りのコツ

1. **最初に結論を言う**
   - ❌ 「今日は、あの、えーと、新しい製品について...」
   - ✅ 「新製品のご紹介です」

2. **1文は短く、シンプルに**
   - ❌ 「この製品は革新的な技術を採用しており、日常生活を便利にし、多くの方に喜ばれています」
   - ✅ 「この製品は、日常生活を便利にします。」

3. **具体的な数字や例を入れる**
   - ❌ 「たくさんの方に使われています」
   - ✅ 「10万人以上が使っています」

4. **最後に行動を促す**
   - 「今すぐダウンロード」
   - 「詳しくは公式サイトへ」
   - 「お問い合わせください」

### デザインのコツ

1. **色は3色まで**
   - メインカラー（背景）
   - アクセントカラー（強調したい部分）
   - 文字色

2. **余白を大切に**
   - 画面いっぱいに文字を詰め込まない
   - 周りに余裕を持たせる

3. **動きは控えめに**
   - 派手すぎるアニメーションは疲れる
   - スムーズでシンプルな動きがプロっぽい
