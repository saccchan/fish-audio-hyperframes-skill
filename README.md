# Fish Audio × HyperFrames スキル

Fish Audio APIで生成した高品質音声をHyperFramesと統合し、音声同期アニメーション動画を作成するCursorスキルです。

**🎯 台本を書いて、Cursorに頼むだけで、プロ品質のナレーション付きアニメーション動画が完成！**

## ✨ 主な機能

### 🎤 高品質音声生成（Fish Audio）
- プロの声優のような自然な音声
- 40言語以上、複数の声質に対応
- 感情表現も可能

### 🎨 美しいアニメーション（HyperFrames）
- 音声に完全同期したテキスト表示
- スムーズなシーン遷移
- プロフェッショナルなエフェクト

### 🚀 自動化
- 台本を書くだけで全自動
- タイミング調整も自動
- レンダリングまで一気通貫

## 🌟 このスキルの特徴

✅ **非エンジニアでも使える** - プログラミング不要  
✅ **日本語で指示** - 専門用語不要  
✅ **完全自動化** - 台本を書くだけ  
✅ **プロ品質** - 高品質な音声とアニメーション  
✅ **何度でも修正可能** - 失敗しても大丈夫  
✅ **どこでも使える** - YouTube、SNSなど自由に投稿
