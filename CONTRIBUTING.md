# コントリビューションガイド

このプロジェクトへの貢献に興味を持っていただきありがとうございます！

## 🤝 貢献の方法

### バグ報告

バグを見つけた場合は、以下の情報を含めてIssueを作成してください：

- 問題の説明
- 再現手順
- 期待される動作
- 実際の動作
- 使用環境（OS、Node.jsバージョンなど）

### 機能リクエスト

新しい機能のアイデアがある場合：

1. まずIssueを作成して提案内容を共有
2. コミュニティからのフィードバックを待つ
3. 承認されたらPull Requestを作成

## 📝 Pull Requestのガイドライン

1. **フォークとブランチ作成**
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. **明確なコミットメッセージ**
   ```
   feat: 新機能の追加
   fix: バグ修正
   docs: ドキュメント更新
   style: コードスタイルの修正
   refactor: リファクタリング
   ```

3. **ドキュメント更新**
   - 新機能を追加した場合は、対応するドキュメントも更新してください
   - 必要に応じてQUICKSTART.mdやGUIDE_FOR_BEGINNERS.mdも更新

4. **テスト**
   - 可能であれば、変更内容をテストしてください
   - サンプル動画が正常に生成できることを確認

## 🎨 コーディング規約

- **ドキュメント**: 初心者にもわかりやすい説明を心がける
- **例**: 具体的な例を豊富に含める
- **言語**: 主に日本語、技術用語は英語も併記

## 📧 連絡先

質問やサポートが必要な場合は、Issueを作成してください。
