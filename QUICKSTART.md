# 📺 3分でわかるクイックスタート

## 🎯 ゴール: 台本を書いて、Cursorに頼むだけで動画完成！

---

## 🚀 超簡単3ステップ

### 【ステップ1】台本を書く（2分）

メモ帳に、話したい内容を書きます。

**台本の例:**

```
こんにちは、新しいサービスのご紹介です。
私たちは、あなたのビジネスをサポートします。
まずは無料で試してみませんか？
詳しくは公式サイトをご覧ください。
```

**ポイント:**
- ✅ 1行 = 1つの場面
- ✅ 1行は2〜8秒で読める長さ
- ✅ 話し言葉で自然に

---

### 【ステップ2】Cursorに頼む（1分）

Cursorを開いて、こうコピペ↓

```
Fish Audioで動画を作ってください。

台本:
（ステップ1で書いた台本をコピペ）

声: 明るい女性の声
デザイン: 暗い背景に明るい文字
動き: スムーズに場面転換
```

**それだけです！**

---

### 【ステップ3】待つ（3〜5分）

AIが自動で:
- ✅ プロの声優みたいな音声を生成
- ✅ 文字アニメーションを作成
- ✅ 背景エフェクトを追加
- ✅ 動画ファイルを出力

**あなたは何もしなくてOK！**

## 🎨 こんなこともできます

### 声を変える

```
Cursorに:
「男性の声に変えて」
「もっと落ち着いた声にして」
```

### 色を変える

```
Cursorに:
「背景を青系に変更」
「文字を赤くして」
```

### 動きを変える

```
Cursorに:
「文字が回転しながら出てくるように」
「もっとゆっくり切り替わるように」
```

### BGMを追加

```
Cursorに:
「明るいBGMを小さく流して」
```

## 🆘 困ったら

### Q: プログラミングがわからない...
**A: 必要ありません！** Cursorに日本語で話すだけ。

### Q: 音声がおかしい
**A: Cursorに「もう一度音声を作って」** と言えばOK

### Q: デザインが気に入らない
**A: Cursorに「背景を〇〇に変えて」** と具体的に伝える

### Q: やり直したい
**A: Cursorに「最初から作り直して」** と言うだけ
