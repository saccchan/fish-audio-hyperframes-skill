## 🎯 台本の構造（わかりやすい例）

### ❌ 悪い台本

```
えーと、今日はですね、新しい製品について話そうと思います。
それで、この製品なんですけど、いろいろな機能があって、
価格もそこそこでして、興味がある方は公式サイトを見てください。
```

**問題点:**
- ❌ 長すぎる（1つの塊）
- ❌ あいまい（「いろいろ」「そこそこ」）
- ❌ 話し言葉が多い（「えーと」「ですね」）

### ✅ 良い台本

```
新製品のご紹介です。
革新的な機能を3つ搭載しています。
価格は1,980円です。
今すぐ公式サイトをチェックしてください。
```

**良い点:**
- ✅ 1行 = 1メッセージ
- ✅ 具体的（「3つ」「1,980円」）
- ✅ シンプルで明確
- ✅ 行動喚起あり（「チェックしてください」）

## 🔄 アニメーションの種類（動きのイメージ）

### エントランスアニメーション（登場の仕方）

```
1. フェードイン
   [ 透明 ] → [ 半透明 ] → [ 完全表示 ]
   ゆっくり現れる

2. スライドイン
   [ 画面外 ] ----→ [ 画面中央 ]
   横から滑り込んでくる

3. ポップアップ
   [ 小さい ] → [ 拡大 ] → [ 通常サイズ ]
   飛び出してくる感じ

4. 回転イン
   [ 回転中 ] → [ 回転停止 ] → [ 表示 ]
   くるっと回りながら登場
```

### シーン遷移（場面転換）

```
1. クロスフェード
   [ シーン1 ] -徐々に- [ シーン2 ]
   ゆっくり溶け合う

2. ワイプ
   [ シーン1 ] ←←← [ シーン2 ]
   左から右へスライド

3. ズームアウト/イン
   [ シーン1 拡大 ] → [ シーン2 登場 ]
   カメラが引く感じ
```

**こう依頼:**
```
動き: 文字が下からスライドイン、クロスフェードで場面転換
```
